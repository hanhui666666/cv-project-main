# 需求文档 - 工业缺陷检测项目

## 1. 项目概述

### 1.1 项目背景
工业产品表面缺陷检测是制造业质量控制的关键环节。传统人工检测方式存在效率低、准确率不稳定、劳动强度大等问题。本项目基于深度学习技术，构建自动化的工业缺陷检测系统，实现对铝型材等工业产品表面缺陷的快速、准确检测。

### 1.2 项目目标
- 构建基于YOLOv8的工业缺陷检测基线模型
- 探索多种改进方向以提升检测性能
- 实现模型可解释性分析（Grad-CAM可视化）
- 完成跨数据集迁移学习验证

## 2. 功能需求

### 2.1 核心功能
| 编号 | 功能模块 | 描述 | 优先级 |
|-----|---------|------|-------|
| F01 | 模型训练 | 支持配置化的模型训练流程 | P0 |
| F02 | 模型评估 | 在验证集上评估模型性能（mAP、Precision、Recall） | P0 |
| F03 | 推理预测 | 对单张图片或视频进行缺陷检测 | P0 |
| F04 | 可解释性分析 | 使用Grad-CAM生成热力图，展示模型关注区域 | P1 |
| F05 | 跨域迁移 | 在不同数据集间进行迁移学习 | P1 |

### 2.2 改进方向（10个实验方向）
| 编号 | 方向 | 描述 |
|-----|------|------|
| 01 | 基线模型 | YOLOv8s基线训练 |
| 02 | CBAM注意力 | 在Backbone中加入CBAM注意力模块 |
| 03 | ECA注意力 | 在Backbone中加入ECA注意力模块 |
| 04 | 损失函数改进 | 使用Focal Loss + WIoU损失 |
| 05 | Neck改进 | 使用BiFPN替代PANet |
| 06 | 数据增强 | 探索Mosaic、MixUp等增强策略 |
| 07 | 模型对比 | RT-DETR与YOLOv8对比实验 |
| 08 | 训练策略 | SWA随机权重平均 |
| 09 | 跨域迁移 | 铝型材→布匹数据集迁移 |
| 10 | 可解释性 | Grad-CAM热力图可视化 |

## 3. 非功能需求

### 3.1 性能要求
- 检测速度：≥ 30 FPS（单张GPU）
- mAP50：≥ 85%（铝型材数据集）
- 模型大小：≤ 50MB

### 3.2 代码规范
- 遵循PEP 8 Python编码规范
- 所有公共函数需添加类型注解和文档字符串
- 配置与代码分离，所有超参数通过YAML文件管理

### 3.3 可维护性
- 模块化设计，各功能组件独立
- 完善的日志记录
- 统一的训练/评估/推理入口

## 4. 数据需求

### 4.1 数据集说明
- **铝型材数据集**：主要训练数据集，包含5类缺陷
- **布匹数据集**：跨域迁移验证数据集

### 4.2 数据格式
- 图像格式：JPG/PNG
- 标注格式：YOLO TXT格式
- 数据划分：训练集80%，验证集20%
